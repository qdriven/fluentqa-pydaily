## What is mimtproxy and where to use it

## Install mimtproxy

- 安装mimtprox,或者建立虚拟环境,进行安装mimtproxy
```shell
pip install mimtproxy
```
## mimmtproxy 启动命令

以下是mimtdump的启动命令:
```shell
mimtproxy
mimtdump
mimtweb
```
我们选择```mimtweb``` 来介绍mitmproxy的使用.
- 启动:
```shell
mimtweb
```
